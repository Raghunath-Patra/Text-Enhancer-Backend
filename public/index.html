Text Enhancement API

This is a serverless API for text enhancement using OpenAI. The API provides endpoints for user authentication and text processing.

Available Endpoints

POST /api/auth/signup

Create a new user account

Body: {"email": "user@example.com", "password": "password123"}

POST /api/auth/signin

Sign in an existing user

Body: {"email": "user@example.com", "password": "password123"}

POST /api/enhance-text

Enhance text using AI (requires authentication)

Headers: Authorization: Bearer <token>

Body: {"text": "Your text here", "enhancement_type": "general"}

Enhancement Types

  • general - General improvement
  • professional - Professional tone
  • casual - Casual tone
  • concise - Make it concise
  • detailed - Add more detail

Authentication

Most endpoints require a Bearer token in the Authorization header. Get your token by signing in through the /api/auth/signin endpoint.

Rate Limits

Users have daily token limits based on their subscription plan. Free users get a limited number of tokens per day.
